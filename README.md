# AdenClient 简介

> AdenClient 是一个社区支持的 2D 沙盒游戏，开发者可以制作高度自定义的模组，创造自己的世界。

正在进行任务：

+ [ ] 使用 SDL2 等库实现游戏框架搭建
+ [ ] 开发基于 Lua 的脚本系统
+ [ ] 开发官方示例模组，设定集 [见此处](doc/DemoModDev-Ver.0.0.1.md)
